Fox yelling 隐私权政策

最后更新：2025年7月3日

一、概述

Fox yelling 是一款 Chrome 浏览器扩展，用于检测用户在使用 AI 平台时是否输入了敏感数据（如手机号、身份证号、银行卡号、API Key 等），并给予实时警告。我们高度重视用户隐私。

二、数据收集

Fox yelling 不收集、不传输、不存储任何用户在 AI 平台中输入的内容。所有敏感数据检测均在用户本地设备上完成，检测结果不会离开用户设备。

三、本地存储

扩展使用 Chrome storage API 在本地存储以下信息：

  • 检测规则缓存（从我方服务器拉取的 JSON 配置，不含用户数据）
  • 用户偏好设置（如警告级别、功能开关状态）
  • 匿名触发统计计数（仅本地累计，不上传）

四、网络请求

扩展仅向以下地址发起网络请求，用于拉取最新检测规则：

  • https://xhslaoren.com/api/v1/rules/*

所有请求中均不包含用户个人信息或任何输入内容。

五、权限说明

  • activeTab：用于判断当前页面是否为受支持的 AI 平台
  • tabs：用于监听页面切换，自动启用或停用防护
  • storage：用于本地保存规则缓存和用户设置
  • alarms：用于定期更新检测规则配置
  • 主机权限：用于在指定 AI 平台页面注入检测脚本

六、第三方共享

我们不会将任何数据出售、出租或以任何形式共享给第三方。

七、政策更新

如本政策发生变更，我们将在本页面更新日期并公布新内容。建议定期查阅。
